# CodeArena

A Cloud-Based Coding Contest Platform built for coding contests and placement preparation.

## 🚀 Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Spring Boot
- Java 21

### Database
- MySQL
- Redis

### Cloud
- Docker
- Kubernetes
- AWS EC2
- AWS S3

## Features

- User Authentication
- Coding Contests
- Problem Management
- Online Code Execution
- Leaderboard
- Admin Dashboard
- Real-time Rankings
